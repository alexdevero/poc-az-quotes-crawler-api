# POC: AZ Quotes crawler

POC for unofficial crawler for AZ Quotes.

## Start dev server

```bash
yarn start

# or
npm run start
```

## Start without watching

```bash
yarn start:ts

# or
npm run start:ts
```

## Compile to JavaScript

```bash
yarn build

# or
npm run build
```
